1. 设置git上的ssh秘钥
在git bash here中输入：
ssh-keygen -t rsa -C "303295018@qq.com"

2. 判断ssh是否配置好
ssh -T git@github.com

3.输入账号
git config --global user.name "breimanbzx"
git config --global user.email "303295018@qq.com"

4.在本地与仓库之间上传
（1） 第一次
git init   # 在本地代码里创建git中介
git add .   # 把所有文件都放到隐藏文件git里
git commit -m "first"  # 代码提交到git
# 关联远程库：origin 为远程库名字
git remote add origin git@github.com:breimanbzx/TheWayOfImage.git   # 连接云端的git仓库：网址是你指定的仓库
# 上传前先把原始版本拉下来
git pull --rebase origin main
# 连接成功后，上传
git push -u origin main
（2）之后更新
git commit -m "修改备注"
git push origin main

5. 异地克隆，修改上传
（1）将路径调到你想要放项目的路径
git clone git@github.com:breimanbzx/TheWayOfImage.git
（2）上传之前先拉一下，以免别人修改
git pull origin main
git commit -m "first"
git push origin main
